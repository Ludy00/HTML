HTML

O que é?

HTML (acrônimo para a expressão inglesa HyperText Makerup Language, que significa Linguagem de Marcação de hipertexto) é uma linguagem, de marcação utilizada para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:

Este curso ensima desde os conceitos básicos da linguagem HTML, até a criação de fórmulários para a Web, ideal para iniciantes, ou para quem quer um guia de consulta, também possui Glossário com todas as palavras mais utilizadas na internet.

O que é?

O Macromedia Dreamwaver é um software de criação para a web criada pela Macromdia (agora Adobe System), que está atualmente na versão CS3. Versões iniciais da aplicação serviam como um simples editor HTML WYSIWYG(what you see is).
